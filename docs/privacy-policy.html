Privacy Policy — Skip Wait

Last updated: February 2025

Overview

Skip Wait is a browser extension that helps you skip countdown timers and waiting pages on supported link shorteners and file hosts. This policy describes what data the extension uses and does not use.

Data collection

Skip Wait does not collect, store, or transmit any personal data or personally identifiable information. We do not run analytics, track users, or maintain user accounts.

Network requests

The extension fetches a public configuration file from Pastebin (a third-party service) to know which sites and scripts to apply. This request does not include any user identifiers or personal data. Content scripts may run on and interact with third-party sites (e.g. link shorteners); any cookies or local storage on those sites are governed by those sites’ own policies, not by Skip Wait.

Permissions

We use Chrome’s scripting and host_permissions only to inject content scripts on the domains defined in the config, so the extension can modify waiting pages. We do not use these permissions to read your browsing history, log URLs, or send your data elsewhere.

Sharing and disclosure

We do not share or sell any user data because we do not collect any.

Changes

If this policy changes, we will update the “Last updated” date. Continued use of the extension after changes constitutes acceptance of the updated policy.
